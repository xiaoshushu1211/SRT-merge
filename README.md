# SRT-merge

# SRT 中英字幕合并小工具

一个轻量级的 SRT 字幕合并脚本：通过弹窗选择中文字幕 `.srt` 和英文字幕 `.srt`，按时间轴逐条合并为一个新字幕文件，并自动输出到同目录下。

## 功能特点

* **图形化选文件**：使用 Tkinter 文件选择框，无需手动输入路径
* **中英逐条合并**：按条目顺序将字幕内容合并为两行（中文在上，英文在下）
* **自动输出文件名**：以译文文件名为基准，输出文件为： `xxxTT.srt`
* **编码兼容**：读取使用 `utf-8-sig`（可兼容带 BOM 的 UTF-8）

## 使用方式

1. 安装 Python 3（系统自带 Tkinter 的常见发行版一般无需额外安装）
2. 将脚本保存为 `merge.py`，或者直接下载merge.py
3. 运行：

```bash
python merge.py
```

4. 按提示分别选择：

   * 译文字幕文件（`*.srt`）
   * 原文字幕文件（`*.srt`）
5. 程序会在中文文件所在目录生成输出文件：`xxxTT.srt`

## 输出格式示例

合并后每条字幕结构如下：

```
1
00:00:01,000 --> 00:00:03,000
中文台词
English line
```

## 注意事项

* 当前版本默认 **按顺序一一对应合并**，因此**两份字幕条目数应一致**；若条目数量不一致，多余部分会被忽略。
* 合并时使用中文文件的时间轴（以中文为准）。

## 适用场景

* 制作双语字幕（学习语言/字幕组自用）
* 给视频播放器提供译文原文同屏字幕版本
